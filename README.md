# GSIS and SSS Compensation Analysis

### Business Task
Analyze trends in Employees’ Compensation (EC) claims, benefits paid, and contributions collected by Government Service Insurance System (GSIS) and Social Security System (SSS) from 1975 to 2023 to assess the financial sustainability of the EC program.

### Data Set
- This dataset provides a historical overview of Employees’ Compensation (EC) program performance of 1975 to 2023 from [Open Data Philippines](https://data.gov.ph/index/public/resource/compendium-of-employees'-compensation-%28ec%29-statistics/number-of-ec-claims,-amount-paid-and-contribution-collected-from-gsis-and-sss-%281975-june-2023%29/zl21lq6x-0aq8-njpa-btd5-f5cuvab7wpfq)

### Tools
- Microsoft Excel for data cleaning and visualization
  - Slicers
  - Pivot Tables / Charts
